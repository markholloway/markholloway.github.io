Autopilot is Twilio's artificially intelligent platform for Voice Bots, SMS, WhatsApp, Facebook Messenger, IVR, Slack, Alexa, and Google Assistant, while using natural langauge understanding and machine learning frameworks.

Everything required to build an Autopilot Assistant is provided in the Twilio Console. If you're like me and get excited anytime you can build someting using command line tools, you'll want to check out the Autopilot CLI. Once installed you will be able to create, update, and delete Autopilot Assistants as well as export an Assistant to JSON.

Autopilot CLI is installed using npm

bash sudo npm install -g @twilio/autopilot-cli
